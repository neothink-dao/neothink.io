# Hub Platform (go.neothink.io)

## Overview

The Hub platform serves as the central integration point for Superachievers in the Neothink ecosystem. It provides holistic growth tools, knowledge management, and cross-platform insights.

## Core Features

### Current Features
- User dashboard with platform integration
- Knowledge repository system
- Profile management
- Cross-platform navigation
- Administrative tools

### Planned AI Features
- Personal AI assistant for growth insights
- Content summarization and recommendations
- Performance analytics
- Intelligent task prioritization

## Technical Stack

- **Framework**: Next.js (Pages Router)
- **State Management**: Platform Bridge
- **Database**: Supabase
- **AI Integration**: OpenAI GPT
- **Authentication**: Supabase Auth

## Directory Structure

```
apps/hub/
├── components/     # Reusable UI components
├── pages/         # Next.js pages and API routes
├── styles/        # CSS and styling
├── lib/          # Utility functions and helpers
├── public/       # Static assets
└── tests/        # Test files
```

## Getting Started

1. **Local Setup**:
   ```bash
   cd apps/hub
   npm install
   ```

2. **Environment Variables**:
   ```bash
   # .env.local
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
   OPENAI_API_KEY=your_openai_key
   ```

3. **Development**:
   ```bash
   # From root directory
   turbo run dev --filter=hub
   ```
