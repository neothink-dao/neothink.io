# Neothink Hub

The central platform for Neothink content, hosted at [go.neothink.io](https://go.neothink.io).

## Features

- Centralized content repository
- User authentication and profiles
- Integration with other Neothink platforms

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Deployment

This app is automatically deployed via Vercel when changes are pushed to the main branch.

## Environment Variables

See the root `.env.example` file for required environment variables.

## Architecture

- Next.js 14.1+
- Tailwind CSS 4.1+
- Shadcn/ui components
- Supabase backend

## Integration

This app integrates with:
- Ascenders platform
- Neothinkers platform
- Immortals platform
